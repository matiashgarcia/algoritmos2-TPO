# TPO Algoritmos II

Implementacion de TDAs y metodos externos para el Trabajo Practico Obligatorio de Algoritmos II.

## Contenido

- `consigna/`: enunciado del TPO.
- `especificaciones/`: interfaces de los TDAs propios.
- `implementaciones/`: implementaciones concretas de los TDAs.
- `metodos_externos/`: resolucion de ejercicios usando TDAs de la catedra.
- `lib/`: libreria provista por la catedra.
- `tests/`: espacio para pruebas manuales.
